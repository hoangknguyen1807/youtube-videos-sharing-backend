## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Typeorm Helper

##### Run migration first
```shell script
# create migration
npm run typeorm:create-migration create-user-table
```

```shell script
# generate migration
npm run typeorm:run-migrations
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
